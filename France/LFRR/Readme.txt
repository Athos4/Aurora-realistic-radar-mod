!!! INSTRUCTION D'INSTALLATION !!!

	- Placer le dossier LFRx ou CRNA xxxx dans le dossier Sectorfile\Include\FR\ puis la FIR correspondante
	- Placer le fichier LFxx.isc dans le dossier Sectorfile


Pour ouvrir le fichier secteur:

	- Dans la fenêtre Aurora "PVD" puis "SECTORS", ouvrir le fichier approche / centre correspondante
			(cette étape charge le secteur avec un profil basique et les couleurs associées)
	- Séléctionner la position souhaitée dans la roue crantée de la fenêtre de connexion lors de la première connexion
			(cette étape charge un profil spécifique à la position)

Le profil intégré inclus:
	
	- Codes SSR
	- Couleurs labels
	- Labels réalistes
	- Filtres traffic hors secteur si équipé
	- Paramètrages radar/minima CA et STCA en fonctions des procédures locales
	- En approche, un menu CPDLC customisé contenant un menu pour des clairances départ via CPDLC ou PDC (ACARS)
	- Filtres positions adjacentes dans le menu ATC

Certains paramètres ne sont pas forcé par le profil, et garde leur valeur de profil précédément chargé et peuvent être éditées dans Aurora
